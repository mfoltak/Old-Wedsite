Chrome Extension

Side Project

GOAL: Created a modern Chrome Extension with Manifest V3 and Pieces IDE application. The Chrome Extension acts as a YouTube bookmarker that allows you to bookmark a specific timestamp in any video and return to your desired timestamp.

IMPLEMENTATION:

CONTRIBUTION: Once you are brought to a YouTube page, if you click on the Chrome Extension Icon you will have the ability to add any number of desired timestamps with a title for each time. You also have the power to delete timestamps or will receive a message if no timestamps are present for the specific YouTube video. If you are not on a YouTube tab, the extension will let you know that it is unable to perform if you click on it.
